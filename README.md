#

## 專案初始化

```bash
$ mkdir ToyVisionPay_project
 # 建立專案資料夾
$ cd ToyVisionPay_project # 進入該專案資料夾
$ git init . # 在本地數據庫-建立空儲存庫

$ mkdir web_server
$ python3 -m venv web\_server/
$ source web\_server/bin/activate
$ cd web_server
$ touch server.py
$ flask --app server run --debug #runserver 你要在web_server底下執行

```
